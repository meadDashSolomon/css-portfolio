Latest Project

Strive Life

This fitness tracking app is an all-encompassing solution for personalized fitness management. It features a secure login system for user data protection, a dynamic Workout Planner integrated with an exercise API for tailored routines, and a Progress Tracker for visualizing fitness achievements. The app also encourages community engagement through a Chat feature and Social Feed. Finally, it includes an AI Chat powered by Open AI's API for customized workout advice. The app's repository is available here for further exploration.

  • Typescript
  • Postgres
  • Tailwind CSS

Legal Research Project

Patent Scanner

Patent Scanner is a tool developed to streamline the process of retrieving, analyzing, and formatting the locations of patent assignees and inventors. This tool operates at a rate approximately 500 times faster than manual processing and with greater accuracy, eliminating a common source of human error. The information retrieved is critical for testing the hypothesis that patent holders located outside the United States are less likely to defend their patents when challenged as invalid. The repository is available here.

Technologies used include:

  • node.js
